# Quản Lý Hàng Hóa

## Thành viên thực hiện
- Phan Thanh Phúc

## Mô tả đề tài
Phần mềm quản lý hàng hóa giúp quản lý:
- Hàng hóa
- Nhà cung cấp
- Tìm kiếm thông tin hàng hóa
- Thêm hàng hóa
- Sửa hàng hóa
- Xóa hàng hóa

## Công nghệ sử dụng
- GitHub
- Jira
- Docker Hub
- SQL Server

## Chức năng hệ thống

### Hiển thị hàng hóa
Hiển thị toàn bộ danh sách hàng hóa.

### Tìm kiếm hàng hóa
Tìm theo tên hàng và đơn giá.

### Thêm hàng hóa
- Không cho thêm trùng mã
- Đơn vị tính: kg/thùng/túi

### Sửa hàng hóa
Cập nhật thông tin hàng hóa.

### Xóa hàng hóa
Xóa hàng hóa khỏi hệ thống.

## Thành phần hệ thống
- Quản lý hàng hóa
- Quản lý nhà cung cấp
- Tìm kiếm dữ liệu
